### Java Association

In java , association was accomplished using IS-A,HAS-A relationships.

1.IS-A relationship is same as extends (.i.e. inheritance)
(weakest)
2. HAS-A relationship divided into 2 categories.
- Aggregation(weaker)
- Composition (strong)
